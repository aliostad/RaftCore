CalculateBroadcastTimeMS()

Calculates the broadcast time: That is, the average time it takes a server to:

  1. send RPCs in parallel to every server in the cluster
  2. receive their responses.
The method uses the node's RaftCore.RaftNode.TestConnection The broadcast time will always be at least 25 ms.
Declaration

Type Description
System.Int32

The estimated broadcast time of the cluster, or 25. Whichever is higher.

CalculateElectionTimeoutMS()

Randomly calculates the election timeout for a node in the cluster. This calculation is based on the return value of CalculateBroadcastTimeMS() The election timeout will be a random number between 12 and 32 times that of the broadcast time.
