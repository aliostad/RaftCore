Apply(String)

Parses the command and applies it to the state machine. This implementation recognises two commands:

  • SET X Y: Adds a key-value pair (X,Y) to the dictionary
  • CLEAR X: Removes the element with key X from the dictionary.
The commands (set/clear) are case insensitive. The keys represented with X above are case sensitive. It silently ignores bad requests.
